ABOUT RTB

WEBSITE

RTB, 'RAISE THE BAR', is a free web application for knowledge-based fun and entertainment. The website theme is a highly customised version of the Azia Admin html template by bootstrapdash. The core functionality of RTB is written in the Python coding language using FLASK framework, also making use of Javascript and other web technologies.

PRIVACY

The application obtains the information you provide during registration. In addition, the application may collect certain information automatically, including, but not limited to, the way you use the application. Any of your information, including passwords that are encrypted, will never be shared with any third party or external service.
